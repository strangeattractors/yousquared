For contributors

Run locally: python3 -m http.server 8000 from this directory, then open http://localhost:8000. A static server is required because browsers block loading local JS files over file://.

Add an industry: drop an entry in shared/industries.js (INDUSTRY_TREE for the picker + INDUSTRY_PREFILLS for the defaults). See README.md.

Structure: each screen is a self-contained HTML file <400 lines. Shared design tokens, iPhone chrome, and React components live in shared/.
